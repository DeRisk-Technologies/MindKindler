# MindKindler Platform User Manual (v1.0)

## Overview
MindKindler is a Clinical Operating System for Educational Psychologists (EPPs) and Local Authorities (LAs). It automates the workflow from Intake to Statutory Reporting (EHCP).

## 1. The Case Workbench
The central hub for managing a child's case.
**Navigation:** Dashboard -> Active Caseload -> Select Case.

### Tabs:
1.  **The Brief:** Shows the "Contract" details (Client, Deadline, Service Type) and the Statutory Timeline status (RAG Rating).
2.  **Case File (Forensic):** 
    *   **Timeline:** Chronological view of all uploaded evidence (PDFs, Emails).
    *   **Stakeholders:** Detected contacts (Parents, SENCOs). You can add new stakeholders here.
    *   **Upload:** Drag & Drop new evidence (Zips, PDFs). The "Clerk Agent" AI automatically indexes them.
3.  **Work Schedule:** 
    *   A To-Do list for the case.
    *   **Engagement Plan:** Track tasks like "School Observation" or "Parent Consultation".
    *   **Calendar:** View upcoming appointments linked to this case.
4.  **Evidence Lab:**
    *   **Consultation:** Launch the "Live Cockpit" to record and transcribe interviews. View past session analysis.
    *   **Observation:** Digital forms for classroom observations (Time sampling, Behavior logs).
    *   **Direct Work:** Launch standardized assessments like **WISC-V**, **WIAT-4**, or **BAS3**.
5.  **Reporting:**
    *   **Report Builder:** Generate statutory drafts (Appendix D, Section F) using AI.
    *   **Templates:** Select from "Statutory Advice", "Intervention Plan", or "Tribunal Witness".

## 2. Live Consultation Cockpit
A real-time tool for recording sessions.
**Features:**
*   **Transcription:** Converts speech to text instantly.
*   **AI Co-Pilot:** Suggests follow-up questions every 15 seconds based on the conversation.
*   **Safeguarding:** Detects risk keywords (e.g., "harm", "abuse") and triggers a "Critical Protocol" alert.
*   **Synthesis:** After the session, use the "Post-Session Synthesis" to generate a summary and intervention plan.

## 3. The Guardian (Compliance)
The background monitoring system.
*   **Breach Radar:** Warns you if a statutory deadline (20-Week) is approaching.
*   **Pattern Detection:** Identifies clusters of risk (e.g., multiple self-harm reports from the same school).
*   **Policy Enforcement:** Blocks actions if consent is missing (e.g., "Cannot finalize report without signed Consent Form").

## 4. How To...
*   **Start a New Case:** Go to Dashboard -> "New Intake +". Upload the "Request for Advice" PDF. The AI will extract the student details.
*   **Run a WISC-V:** Go to Case Workbench -> Evidence Lab -> Direct Work -> Click "WISC-V". Enter the raw scores.
*   **Draft a Report:** Go to Case Workbench -> Reporting -> Select Template -> Click "Generate Draft".
*   **Add a Parent:** Go to Case Workbench -> Files -> Stakeholders -> Click "+".

## 5. Partner Portal (Enterprise)
For LAs and Agencies.
*   **Commissioning:** LAs can "Push" cases to Independent EPPs.
*   **Deal Registration:** Partners can log new opportunities via the Partner Portal.
*   **Certification:** Track your partner status (Silver/Gold).
